# OLA_Ride_Performance_Revenue_Analysis_July


## Overview
This project analyzes OLA ride data for July 2024 to uncover insights related to booking performance, revenue distribution, cancellations, vehicle efficiency, and service quality.

## Objectives
- Analyze booking success and cancellation trends
- Understand revenue contribution by payment method and vehicle type
- Evaluate operational efficiency and ride distance patterns
- Assess driver and customer ratings

## Tech Stack
- PostgreSQL
- SQL
- Power BI
- CSV Data Source

## Dataset
- `july.csv` – Raw ride-level data
- `SQL_Queries.sql` – SQL queries for data analysis

## Dashboards
- Overall performance overview
- Vehicle type analysis
- Revenue and payment insights
- Cancellation breakdown
- Ratings analysis

## Key Insights
- Significant cancellation rates impact overall success
- Customer-driven cancellations dominate
- Cash and UPI are primary revenue drivers
- Consistent service ratings across vehicle types

## Limitations
- Single-month analysis
- No predictive modeling
- Static dataset

## Future Enhancements
- Multi-period trend analysis
- Predictive analytics
- Automated ETL pipeline
